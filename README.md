# garina-cs32
Given a vague craving description from the user (e.g. I want something ‘salty and crispy’ or ‘warm and comforting’), we want to use our database of dishes to match the user to one food item that would most satisfy their craving. 

To run the program, python must be installed on the computer and no external libraries or packages are required. Download or clone this repository, navigate to the project folder in terminal, and run the program. After running the script, user will be prompted: "Describe what you're craving: ". Enter a description of craving, such as "warm and comforting" or "sweet and fruity" and the program will output the best matching food, a match score, and a ranked list of all foods with score bars and matched tags.
